# Silithium

Silithium is a hybrid signature combining EC-Schnorr (or ECDSA) with ML-DSA.

The current variant is only the **silithium-44**.

## Resources

This is based on the recent research paper [Compact, Efficient and Non-Separable Hybrid Signatures](https://ia.cr/2025/2059).
